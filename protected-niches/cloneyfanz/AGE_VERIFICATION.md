# CloneyFanz — Age Verification Requirements (Planning Only)

**No implementation permitted.**

## Requirements

- All users must verify age as 18+ before accessing any adult content
- Age verification must use a third-party compliant verification provider
- Verification records must be retained per applicable law
- Re-verification required if access methods change

## Potential Approaches (Planning Only)
- Document-based verification (government ID)
- Credit card verification (billing address + card ownership implies 18+)
- Third-party age verification service integration

## Non-Negotiable
- No self-attested "I am 18+" checkboxes without additional verification
- No access to adult content before verification is complete and confirmed
